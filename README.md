# Ledger — Expense Intelligence

A single-file personal expense dashboard. Dark, fast, installable as an app.

- **Log in seconds** — press `N`, amount, category chip, done
- **Insights engine** — overspend alerts, month-end pace projection, creeping-category detection
- **Budgets** — editable monthly caps per category with live warnings
- **Trends** — 6-month bars, month-to-month time travel
- **Reports** — printable/PDF monthly report + email draft
- **Private by design** — all data stays in your browser's local storage; export CSV/JSON backups anytime

No build, no dependencies, no server. Open `index.html` or serve statically.
